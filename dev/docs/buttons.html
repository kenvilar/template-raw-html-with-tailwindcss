Overview

Buttons are provided as a reusable include at @ui/button.html. You can pass parameters to configure text, href, and className (variant).

Live examples

.btn.primary
.btn.secondary
.btn.tertiary

Quick usage 

<div
  data-include="@ui/button.html"
  data-include-params='{
    "text": "Click me",
    "href": "/get-started",
    "className": "primary"
  }'
></div>

You can also pass params as attributes or via query string: 

<!-- Attributes -->
<div data-include="@ui/button.html" data-include-text="Click me" data-include-href="/get-started" data-include-className="secondary"></div>

<!-- Query string -->
<div data-include="@ui/button.html?text=Click%20me&href=%2Fget-started&className=tertiary"></div>

API

  • text — string. Visible label. Default: "Button".
  • href — string. Link destination. Default: "#".
  • className — string. Visual variant. Supported: "primary", "secondary", "tertiary". Default: "primary".
  • Optional size modifiers: append sm or lg to the class list if you author variants, e.g., className: "primary sm".

Note: If you pass parameters via data-include-params, make sure the JSON is valid. Trailing commas and comments are tolerated by the parser in include.js.
